Open two terminal windows.

1. In the first terminal, run the server code:

		For TCP:
		python tcp_server.py

		For UDP:
		python udp_server.py

2. In the second terminal, run the client code:

		For TCP:
		python tcp_client.py

		For UDP:
		python udp_client.py

3. Enter your message in the client terminal when prompted.

Note : Always start the server before starting the client. Both the server and client must use the same host and port.
